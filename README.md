# TCG Command Engine

Static cinematic experience for Talkerstein Consulting Group.

Production URL: https://tcg-command-engine.vercel.app/

## Structure

- `index.html` contains the inline layout, styling, and animation engine.
- `assets/manifest.js` maps scene keys to local poster and video assets.
- `assets/` contains the generated video, poster, logo, and icon files.

## Deploy

This project is a static Vercel site. Pushes to the connected GitHub repository should deploy from the repository root.
